Malayalam-English Code-Mixed Sentiment Preprocessing

This repository contains the preprocessing pipeline for the Malayalam-English code-mixed sentiment analysis dataset.

Steps performed:

Removed null rows and duplicates

Minimal cleaning (URLs, mentions, whitespace normalization)

Preserved code-mixing and Malayalam script

Kept 5-class sentiment structure

Encoded labels to numeric IDs

Saved clean train/dev/test splits
